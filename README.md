# BMI Calculator 

Hey! This is a super simple, no-nonsense BMI Calculator I built using just the basics: **HTML, CSS, and JavaScript**. No heavy frameworks or fancy libraries here, just clean code to help you figure out where you stand with your weight and height.

You can check out the live version or peek at the code right here on GitHub.

### What it does
Basically, you put in your height and weight, and it spits out your Body Mass Index (BMI). It’s a quick way to see your body health stats and if your weight-to-height proportion is in a good spot. 

*   **Fast Calculations:** Get your result instantly.
*   **Health Categories:** It doesn't just give you a number; it tells you if you're underweight, normal, overweight, etc.
*   **Simple UI:** I kept the design realy clean so anyone can use it without getting confused.

### Why I made this
I wanted to make somthing that was easy to use and actually useful for daily health tracking. Sometimes you just need a quick tool without all the extra bloat. Plus, it was a great way to practice some vanilla JS logic and DOM manipulation.

### How to use
1.  Open the `index.html` file in your browser.
2.  Type in your height (in cm).
3.  Enter your weight (in kg).
4.  Hit calculate and see your results!
